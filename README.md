# 🏛️ Art Hub — Harvard Museum Exploration

A high-performance, minimal, and fully typed web application built to explore the rich archives of the Harvard Art Museums. This project is crafted with a focus on **Zero-Waste Code**, clean architectural patterns, and seamless responsive design.

🔗 **Live Demo: https://art-hub-eight.vercel.app/


---

## 🚀 Key Features & Highlights

- **Smart Dynamic Pagination:** A fluid, defensive 10-button sliding window navigation that prevents layout shifts and guarantees exactly 10 buttons in any boundary page state.
- **Optimized Data Fetching:** Completely stripped of redundant server checks and double-checks, ensuring fast API communication using native protocols.
- **Alphabetical Filtering:** Dynamically maps and filters through 100 world cultures, sorted efficiently from A to Z on the client side.
- **Robust Component Layouts:** Modular card components utilizing strict flexbox boundaries to prevent rendering distortions on varying image heights.

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Custom Premium Museum Palette)
- **API Source:** Harvard Art Museums API
